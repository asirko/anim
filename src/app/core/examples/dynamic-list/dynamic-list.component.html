Liste dynamique

Il s'agit d'animer les éléments d'une liste qui est mise à jour dynamiquement par un observable.

Les éléments peuvent donc individuellement être ajouté ou retiré au cours du temps.

La liste est affiché avec *ngFor="let element of list$ | async".

Exemple

Principe général

Angular met à disposition un état virtuel : void. Cet état peut être vu comme l'élément n'est pas dans le DOM.

Il est alors possible de créer une transition sur son ajout et sa suppression du DOM : void <=> *
